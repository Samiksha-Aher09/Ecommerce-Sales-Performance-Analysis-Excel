# Excel-Annual-Data-Analysis
# Project Overview :- This project is an Excel-based Annual Sales Analytics Dashboard for 2022, created to analyze sales performance, customer behavior, order trends, and channel-wise contribution using advanced Excel features.
  The dashboard provides interactive insights using slicers, charts, and pivot tables to support data-driven business decisions.
# Objectives
  - Analyze monthly order vs sales trends
  - Understand gender-wise sales contribution
  - Track order status distribution
  - Identify top 5 states by sales
  - Analyze age group vs gender purchasing behavior
  - Evaluate sales performance by channels
# Key Insights
  - Women customers contribute 64% of total sales
  - 92% of orders are delivered successfully
  - Amazon contributes 35% of total sales
  - Maharashtra is the highest sales-generating state
  - Adult women represent the largest purchasing segment
  - Sales peak during February and March
# Tools & Excel Features Used
  - Microsoft Excel
  - Pivot Tables & Pivot Charts
  - Slicers (Month, Channel, Category)
  - Advanced Excel formulas (SUMIFS, COUNTIFS)
  - Data cleaning & preprocessing
  - Dashboard design & formatting
# Dashboard Components
  - Order vs Sales (Monthly Trend)
  - Sales: Men vs Women
  - Order Status Distribution
  - Top 5 States by Sales
  - Orders by Age Group & Gender
  - Orders by Sales Channels
# Dataset Information
  - Sales transaction records (2022)
  - Customer demographic details
  - Order status and category data
  - Channel-wise sales information
  - State-wise sales data
# Business Use Case
  - Identify top-performing sales channels
  - Improve order fulfillment efficiency
  - Target high-value customer segments
  - Support strategic planning using data insights
